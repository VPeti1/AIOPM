## AIOPM
An all in one package manager for linux

## How to install
# Download the latest version
# Type in chmod +x ./AIOPM.out
# Type in ./AIOPM.out

## How to build with g++
# Clone the repo
# Edit the code to your liking
# Type in g++ main.cpp

## What are its features?
# Manages apt,yum,pacman and flatpak with (really) easy syntax

# Known issues
